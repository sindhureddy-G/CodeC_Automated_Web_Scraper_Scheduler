# Automated Web Scraper with Scheduler

## CodeC Technologies Internship Task 2

This project is an Automated Web Scraper with Scheduler built using Python. It automatically extracts data from websites and runs at scheduled time intervals.

## Features

* Web scraping using Python
* Automated scheduling of tasks
* Extracts real-time data from websites
* Time-based execution
* Command-line execution

## Technologies Used

* Python
* BeautifulSoup / Requests (if used)
* Schedule / Time module

## How to Run

1. Clone the repository
2. Install required libraries:

   ```
   pip install requests beautifulsoup4 schedule
   ```
3. Run the script:

   ```
   python web_scraper_scheduler.py
   ```
